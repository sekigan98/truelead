# TrueLead Website

Sitio estático inicial para **TrueLead**, listo para subir a GitHub y desplegar luego en Render, Netlify, Vercel o GitHub Pages.

## Incluye

- `index.html` → landing principal
- `dashboard.html` → demo del panel para agencias
- `connect.html` → página de vinculación QR de WhatsApp
- `css/styles.css` → estilos globales
- `js/app.js` → JS básico
- `assets/logo.png` → logo base de TrueLead
- `assets/preview-landing.png` → mockup de referencia
- `assets/preview-dashboard.png` → mockup de referencia

## Cómo usar

1. Subí todo el contenido del ZIP a un repositorio en GitHub.
2. Podés abrir `index.html` directamente o levantarlo con cualquier hosting estático.
3. Para Render más adelante, podemos convertir esto en un proyecto con backend y conexión real a WhatsApp.
